Given a string S consisting only '0's and '1's,  find the last index of the '1' present in it.

Example 1:

Input = 00001

Output:

Enter the value in '0' & '1' : 01110
Last index of  1 in given string is : 3
******************************************
Example 2:

Input: 0
Output:

Enter the value in '0' & '1' : 0000
Since, 1 is not present, so output is : -1
