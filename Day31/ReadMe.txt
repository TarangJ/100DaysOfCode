The reversing of a string is nothing but simply substituting the last element of a string to the 1st position of the string.

Example 1:

Input: understood

Enter the string : understood
Reversed string : dootsrednu
No. of letters in string : 10

***********************************

Input: whatAmIdoingwithmylife007

Output:

Enter the string : whatAmIdoingwithmylife007
